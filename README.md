##  Objetivo  

Este projeto tem como foco analisar os **dados do refeitório universitário da UFC (PRAE)**, explorando padrões de consumo entre os estudantes.  
Foram aplicados conceitos de **porcentagem** e **probabilidade**, com o intuito de compreender o comportamento alimentar discente e gerar insights úteis para a **gestão da assistência estudantil**.  

>  Exemplo de pergunta respondida:  
> “Qual a probabilidade de um discente almoçar e jantar no mesmo dia?”  

---

##  Principais análises  

-  Cálculo de **porcentagens** de alunos que realizam diferentes combinações de refeições (almoço e/ou jantar).  
-  Estimativa de **probabilidades conjuntas e condicionais** com base nos dados da PRAE.  
-  Visualização simples das proporções para facilitar a interpretação.  
-  Discussão sobre possíveis aplicações dos resultados para **planejamento do refeitório** e **redução de desperdício**.
